# LexisNexis GraphQL API

LexisNexis provides legal research, news, and business intelligence data. The API covers legal case search, statute retrieval, news aggregation, company profiles, and Accurint identity data for law firms, enterprises, and government agencies.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from LexisNexis REST API
**Documentation:** https://dev.lexisnexis.com/
